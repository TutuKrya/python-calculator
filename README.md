# Умный калькулятор с историей операций

Простой, но мощный калькулятор с историей операций и обработкой ошибок

## Возможности

- Базовые математические операции (+, -, *, /)
- Дополнительные операции (**, //, %)
- Полная история вычислений с возможностью очистки
- Доступные команды: yes, history, clear, exit
- Обработка ошибок ввода
- Интерактивный режим работы
- Красивый формат ввода

## Использование

```bash
python calculator.py
